## When to Use Advanced Features

### Streaming

Use streaming when:
- Building interactive chat interfaces
- Responses are expected to be long
- You want to show progress to users
- Implementing real-time applications

### Memory & RAG

Use memory and RAG when:
- Conversations need context from previous sessions
- You need to query external knowledge bases
- Building domain-specific assistants
- Processing large document collections

### Multi-Agent

Use multi-agent orchestration when:
- Tasks require different specialized capabilities
- You want to implement review/validation workflows
- Building complex autonomous systems
- Implementing hierarchical agent structures

### Plugins

Use plugins when:
- Adding custom functionality to Ember
- Integrating with external services
- Sharing tools across projects
- Building modular agent systems
